# English-Only Website Collection

A separate collection of the **English-language** scraped websites, built from
`scraped/` by detecting each page's language locally (langdetect) and copying
only English pages into `english_only/`. Finnish and all other non-English
pages are excluded (originals in `scraped/` are left untouched — nothing was
deleted). Generated by `collect_english.py`.

## How many English websites

| Metric | Count | Share of scraped |
|---|---|---|
| Total scraped files | 50,076 | 100% |
| **English websites (clean)** | **10,254** | **20.5%** |
| Excluded as junk (bot-verification / challenge pages) | 7,577 | 15.1% |

### Data-quality note

A first pass detected 17,831 "English" files, but **7,577 of those were
bot-verification / JS-challenge interstitials** (e.g. *"Please wait while your
request is being verified…"*, *"Just a moment…"*, Cloudflare challenge pages)
that the scraper captured instead of real content. They are in English but are
**not** genuine website content, so they are excluded by default. The genuine
English count is therefore **10,254**, not 17,831. (Re-run with `--keep-junk`
to include them.)

## Outputs

* `english_only/<domain>.txt` — the 10,254 English pages
* `english_index.csv` — per-file language label: `domain, lang, chars, english, source_path, english_path`
* `english_only_symbol_counts.csv` — per-file symbol counts for the collection

## Reproduce

```bash
python3 collect_english.py                 # clean collection (junk excluded)
python3 collect_english.py --keep-junk      # include interstitial pages
python3 count_symbols.py --dir english_only
```
