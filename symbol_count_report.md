# Symbol Count Report — Scraped Corpus

Counts of **symbols (Unicode characters)** in every scraped main-page text file.
Symbols are counted as Unicode code points, so Finnish letters such as `ä`/`ö`
count as one symbol each (not their multi-byte UTF-8 encoding). Generated by
`count_symbols.py` over `scraped/`.

## Summary — `scraped/`

| Metric | Value |
|---|---|
| Files | 50,076 |
| **Total symbols (characters)** | **83,293,840** |
| Total bytes (UTF-8) | 85,844,290 |
| Total words | 9,713,280 |
| Total lines | 1,033,106 |
| Empty files | 0 |

### Symbols per file

| Statistic | Symbols |
|---|---|
| Minimum | 1 |
| Median | 669 |
| Mean | 1,663.3 |
| Maximum | 2,154,927 |

Bytes (85.8M) exceed symbols (83.3M) because multi-byte Finnish characters take
more than one UTF-8 byte but count as a single symbol.

The distribution is strongly right-skewed (median 669 vs. mean 1,663, max 2.15M):
a small number of very large pages pull the mean up, so the **median** is the
more representative measure of a typical page.

## Per-file data

Full per-file counts are in **`scraped_symbol_counts.csv`**:

```
domain, path, chars, bytes, words, lines
```

## Reproduce

```bash
python3 count_symbols.py --dir scraped          # this report's numbers
python3 count_symbols.py --dir translated        # English translations (when ready)
python3 count_symbols.py --dir english_only      # English-only collection
```
